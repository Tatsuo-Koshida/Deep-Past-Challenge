# Curated train.csv versions (tracked)

このフォルダに、品質改善した `train.csv` を **バージョンごとに Git 管理**して保存します。

## Naming convention (recommended)

- `train.curated.v001.csv`
- `train.curated.v002.csv`
- ...

（日付ベースにしたい場合）

- `train.curated.2026-03-11.csv`

## Notes

- 公式 raw は `data/kaggle/deep-past-initiative-machine-translation/train.csv`（上書きしない）
- 各バージョンで「何を変えたか」は `.codex/docs/train_csv_curation_log.md` に残す
